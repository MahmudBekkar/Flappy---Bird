# Flappy Bird

A simple **Flappy Bird clone** built using **C# and Windows Forms**.

## 🎮 Game Description

The player controls a bird and must fly between pipes without hitting them.

## 🛠 Technologies

* C#
* Windows Forms
* Visual Studio

## ▶️ How to Run

1. Open `FlappyBird.sln` in Visual Studio
2. Press **Start (F5)** to run the game
